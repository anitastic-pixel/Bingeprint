Import a list export (MAL / AniList XML)

Drop a MyAnimeList XML export — Smart Scoring folds its scores and watch statuses into your taste vector. AniList users with a private list can use this too: every AniList export tool produces this same MAL-XML format, so it imports here all the same. (Public AniList list? The username import in the popup is simpler.)

MyAnimeList export:
  1. Sign in at myanimelist.net.
  2. Open your profile → click the Anime List tab.
  3. In the left sidebar: Export → choose Anime List → click Export My List.
  4. You'll get an animelist_…xml.gz file. Drop it below as-is, or unzip it to .xml first — either works.
AniList export (private lists):
  1. AniList has no built-in export, so use a converter such as anilist-to-mal or malscraper to turn your AniList list into a MAL-format .xml.
  2. Drop that file below — same as a MAL export.
